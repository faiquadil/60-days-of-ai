# Day 36 – Cognitive Pattern Explorer

## Objective

Build and interact with an AI-generated web application that explores different cognitive thinking patterns through interactive scenarios, prioritization activities, and reflective exercises.

---

## Tools Used

* Claude AI
* HTML, CSS & JavaScript
* Web Browser
* GitHub

---

## Workflow Completed

* Read the provided resources.
* Watched the solution video.
* Opened Claude and set the effort level to **Low**.
* Started a new conversation.
* Generated the **Cognitive Pattern Explorer** HTML application.
* Saved the generated HTML file.
* Opened the application in the browser.
* Completed the experience in **Calm Mode**.
* Finished all three chapters:

  * Scenario-based decision making
  * Priorities drag-and-drop activity
  * Decision mapping activity
* Reviewed the personalized Reflection Journal.
* Repeated the complete experience in **Stress Mode**.
* Compared thinking tendencies across both modes.
* Captured screenshots of the final Reflection Journal and Thinking Pattern Snapshot.
* Uploaded the HTML file, screenshots, and documentation to GitHub.

---

## Calm Mode Result

* **Analytical Thinker:** 41%
* **Balanced Reflective Thinker:** 23%
* **Overthinking Loop Style:** 14%
* **Emotional Intuitive:** 12%
* **Action-First Decision Maker:** 10%

### Observation

When calm, my decisions are primarily guided by logic, structured analysis, and thoughtful reflection before taking action.

---

## Stress Mode Result

* **Overthinking Loop Style:** 52%
* **Emotional Intuitive:** 15%
* **Analytical Thinker:** 13%
* **Balanced Reflective Thinker:** 11%
* **Action-First Decision Maker:** 9%

### Observation

Under stress, I tend to revisit decisions repeatedly and seek additional certainty before committing, while emotions play a greater role in my decision-making process.

---

## Key Learnings

* Thinking patterns can change depending on emotional state.
* Calm decision-making encourages logical analysis and structured reasoning.
* Stress may increase uncertainty and overthinking.
* Self-reflection helps identify personal cognitive tendencies.
* Interactive simulations are an engaging way to build self-awareness and improve decision-making skills.
