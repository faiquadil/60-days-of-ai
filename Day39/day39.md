# Day 39 – PDF Splitter & Merger

## Objective
Built and tested a browser-based PDF Splitter & Merger application using Claude.

## Features
- Split PDF into multiple files
- Merge multiple PDFs into one
- Drag-and-drop file upload
- Preview PDF pages
- Reorder pages before merging
- Download processed PDFs
- 100% client-side processing
- Offline support
- Responsive interface
- Dark mode support

## What I Learned
- Prompt engineering can generate complete web applications.
- Client-side PDF processing improves privacy because files never leave the browser.
- JavaScript libraries can efficiently manipulate PDF documents.
- Interactive UI enhances user experience for document management.
- Testing both split and merge workflows ensures application reliability.

## Outcome
Successfully generated, tested, and documented a fully functional PDF Splitter & Merger web application.
