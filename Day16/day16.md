# Day 16 - Stock Fundamental Research Skill

## Objective

Created and tested a custom Claude skill named stock-fundamental-research.

## Skill Description

The skill analyzes Indian and global listed companies using:

* Fundamentals
* Financial statements
* Valuation metrics
* Ownership trends
* Risks
* Growth prospects

## Test Conducted

Compared TCS and Infosys in Deep Dive mode.

## Features Observed

* Company overview
* Valuation analysis
* Revenue and profit trends
* ROE and ROCE analysis
* Ownership pattern
* Risk assessment
* Peer comparison

## Key Learnings

1. Custom skills can automate repetitive tasks.
2. Structured stock research reports can be generated quickly.
3. Comparison reports provide a detailed view of multiple companies.
4. Skills are reusable without re-entering instructions.
