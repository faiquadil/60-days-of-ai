# Day 21 – Digital Privacy Dashboard

## Objective

Generate an interactive Digital Privacy Dashboard using Claude AI to analyze a sample user's digital footprint.

## Tools Used

* Claude AI
* HTML
* CSS
* JavaScript
* GitHub

## Tasks Completed

* Generated a Digital Privacy Dashboard using the provided prompt.
* Saved the generated HTML file.
* Opened the dashboard in a web browser.
* Reviewed the Digital Footprint Score and Privacy Score.
* Analyzed the Exposure Heatmap, Company Exposure Ranking, Risk Radar, Digital Twin Profile, and Privacy Improvement Plan.
* Captured dashboard screenshots.
* Uploaded the HTML file, screenshots, and documentation to GitHub.

## Key Findings

* Total Services Used: 15
* Parent Companies: 5
* Digital Footprint: Significant
* Privacy Score: Fair
* Ecosystem Concentration: High
* Tracking Surface: High

## Learnings

* Learned how AI can generate interactive cybersecurity dashboards.
* Understood how digital services contribute to online exposure.
* Explored privacy scoring, exposure analysis, and digital footprint visualization.
* Learned the importance of reducing tracking and improving online privacy settings.
