# Day 44 – LinkedIn Roast & Rebuild

## 📌 Project Overview
Today I built an AI-powered **LinkedIn Roast & Rebuild** web application using Claude. The application analyzes a LinkedIn profile, identifies weak areas from a recruiter's perspective, rewrites important sections, provides an improvement score, and generates a personalized 7-day action plan to optimize the profile for internships and entry-level roles.

---

## 🎯 Objective
To understand how AI can be used to review and optimize professional profiles while learning how recruiters evaluate LinkedIn profiles for search visibility and hiring.

---

## 🛠️ Features

- 🔍 Complete LinkedIn Profile Audit (Roast)
- ✨ Optimized Headline Suggestions
- 📝 Professional About Section Rewrite
- 💼 Experience Section Improvements
- 🧠 Skills & Keyword Recommendations
- 📊 Before vs After Scorecard
- 📅 Personalized 7-Day LinkedIn Improvement Plan
- 📋 Final Summary Card for Easy Sharing

---

## 💻 Technologies Used

- HTML5
- CSS3
- JavaScript
- Claude AI

---

## 📈 Results

### Overall Score

- **Before:** 31/100
- **After:** 82/100
- **Improvement:** +51 Points

### Section-wise Improvements

| Section | Before | After |
|----------|---------|--------|
| Headline | 3/10 | 8/10 |
| About (Hook) | 2/10 | 8/10 |
| About (Full) | 2/10 | 9/10 |
| Experience | 4/10 | 8/10 |
| Skills | 5/10 | 8/10 |

---

## 📚 Key Learnings

- Recruiters spend only a few seconds scanning a LinkedIn profile.
- Strong headlines should focus on role-specific keywords rather than generic words like *Aspiring* or *Exploring*.
- The About section should demonstrate achievements and value instead of only stating interests.
- Experience descriptions become more impactful when written with action-oriented language.
- Proper keyword optimization improves LinkedIn search visibility.
- AI can significantly speed up profile optimization while maintaining professional quality.
