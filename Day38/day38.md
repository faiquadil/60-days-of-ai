# Day 38 – Typing Speed Studio

## 📌 Overview
On Day 38 of the ABTalks 60-Day AI Challenge, I built **Typing Speed Studio**, an AI-powered typing practice application. The app provides multiple typing modes, adaptive practice categories, and real-time analytics to help users improve typing speed, accuracy, and consistency.

---

## 🎯 Objective
Design an interactive typing practice platform that offers different practice modes, detailed performance tracking, and a smooth user experience.

---

## ✨ Features
- Adaptive typing practice across seven disciplines
- Seven typing modes:
  - Time
  - Words
  - Quote
  - Programming
  - Custom
  - Focus
  - Zen
- Category switching
  - General English
  - Academic
  - Business
  - Medical
  - Legal
  - Creative Writing
- Multiple word counts (25, 50, 100, 250)
- Live typing statistics
- WPM & Raw WPM tracking
- CPM calculation
- Accuracy monitoring
- Consistency score
- Character breakdown
- WPM progress graph
- Accuracy over time
- Error heatmap
- Achievement system
- Performance summary
- Dark UI with responsive design
- Keyboard shortcuts
