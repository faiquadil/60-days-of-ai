# Day 33 – Media Integrity Analyzer

## 📌 Project Overview

Today I developed **Media Integrity Analyzer**, an interactive web application designed to teach media literacy through guided discovery instead of traditional quizzes. The application helps users identify misleading headlines, emotional manipulation, and persuasive writing techniques commonly used in modern digital media.

The entire application is built as a **single offline HTML file** using Vanilla HTML, CSS, and JavaScript without relying on external libraries or APIs.

---

## 🎯 Objectives

- Build an interactive educational experience.
- Teach users how misleading headlines work.
- Demonstrate emotional manipulation techniques.
- Improve critical thinking before consuming or sharing online content.
- Generate fictional scenarios for safe learning.

---

## 🚀 Features

### Theme & Interface
- Editorial-inspired dark interface
- Responsive layout
- Smooth animations
- Progress tracking
- Interactive lesson flow

### Challenge 1 – Headline Detective
Users learn how headlines can exaggerate information.

Features:
- Fictional news headline
- Matching article
- Click decision (Yes / Maybe / No)
- Identify exaggerated words
- Headline Accuracy Score
- Highlight misleading phrases
- Fair headline rewrite
- Key learning takeaway

### Challenge 2 – Emotion Detector
Users analyze emotionally persuasive content.

Features:
- Fictional social media post
- Emotional response selection
- Detect emotionally loaded language
- Target audience analysis
- Manipulation technique explanation
- Neutral rewritten version
- Learning takeaway

### Live Media Integrity Metrics

The application continuously updates:

- Headline Accuracy
- Source Reliability
- Emotional Manipulation
- Audience Targeting

### Final Dashboard

After completing both challenges, users receive:

- Overall Media Integrity Score
- Biggest Red Flag
- Summary of learning
- Three practical media literacy habits
- Replay with new scenarios
