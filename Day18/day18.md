# Day 18 – Brain Dump Action Planner

## Objective

Create and test a reusable Claude Custom Skill that converts unstructured notes into structured dashboards and actionable plans.

## Skill Name

brain-dump-action-planner

## Description

A custom Claude skill designed to transform messy notes, meeting transcripts, brainstorming sessions, and project discussions into organized summaries, action items, risks, blockers, decisions, conflicts, and open questions.

## Test Scenario

Hospital Management System – Project Meeting

### Input Notes

* Ahmed: Database schema should be finalized by Friday.
* Sara: User interface design needs another review.
* Ali: Testing cannot begin until the database is completed.
* Development team wants new features added.
* Testing team wants feature freeze.
* No dedicated QA resource assigned.
* Patient registration module approved.
* Open Question: Who will handle deployment?

### Output Generated

The skill generated:

* Summary
* Key Takeaways
* Action Items Table
* Risks & Blockers
* Conflicts Section
* Decisions Made
* Open Questions
* Interactive Dashboard

## Key Learnings

* Claude Custom Skills can automate note organization.
* HTML dashboards provide a clear project overview.
* Risks, conflicts, and action items can be extracted automatically.
* Reusable skills reduce repetitive prompting.
* Interactive dashboards improve project tracking and decision-making.
