# Day 20 - AI Face Puzzle Game

## Objective

Built an AI Face Puzzle Game using HTML, CSS, and JavaScript with the help of Claude AI.

## Features Implemented

* Image Capture / Upload
* Puzzle Difficulty Levels (3×3, 4×4, 5×5)
* Drag and Drop Interaction
* Touch Support
* Timer Tracking
* Move Counter
* Puzzle Completion Detection
* Leaderboard using Local Storage
* Responsive User Interface

* Working with AI-assisted application development
* Creating interactive puzzle logic
* Image slicing and rendering techniques
* Handling drag-and-drop functionality
* Managing timers and move counters
* Using localStorage for storing leaderboard data
* Building responsive web applications

Successfully developed and tested an AI Face Puzzle Game with multiple difficulty levels and leaderboard functionality.
